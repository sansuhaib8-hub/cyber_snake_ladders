# 🎮 Cyber Snake & Ladders

یاری مار و پەیژەی سایبەری بە Flutter - پڕۆژەیەکی مۆدێرن بە Clean Architecture

## 🌟 تایبەتمەندییەکان

- ✅ 4 یاریزان - یاری بە هاوڕێکان
- 🐍 8 مار - بەرز دەبنەوە و دایبەزێنن
- 🪜 8 پەیژە - دەتگەیەننە سەرەوە
- 🎨 دیزاینی سایبەری - ڕەنگی نیۆن و تاریک
- 🎲 ئەنیمەیشنی زار - rotation و glow
- 🏆 سیستەمی leaderboard - تۆمارکردنی براوەکان
- 💾 پاشەکەوتکردنی یاری - بە shared_preferences
- 🌐 پشتگیری کوردی - RTL

## 🛠️ تەکنەلۆجیا

- Flutter 3.x - فریمۆرکی سەرەکی
- Riverpod 2.4.9 - State Management
- Shared Preferences 2.2.2 - Local Storage
- AudioPlayers 5.2.1 - دەنگ و مۆسیقا
- Google Fonts 6.1.0 - فۆنتەکان

## 📁 پێکهاتەی پڕۆژە

- lib/main.dart - دەستپێکی ئەپ
- lib/app.dart - ڕێکخستنی MaterialApp
- lib/core/ - بنەما و هاوکارەکان
- lib/models/ - مۆدێلەکانی داتا
- lib/data/ - بەڕێوەبردنی داتا
- lib/domain/ - لۆجیکی بزنس
- lib/presentation/ - UI و controllers

## 🚀 دامەزراندن

1. Dependencies دابەزێنە:
   flutter pub get

2. یاریەکە اجرا بکە:
   flutter run

## 🎯 چۆن یاری بکەیت

1. دەستپێکردن - کلیک لە "دەستپێکردنی یاری" بکە
2. هاویشتنی زار - کلیک لە بۆرد یان زارەکە بکە
3. جوڵان - یاریزان بە ئۆتۆماتیکی دەجوڵێت
4. مار و پەیژە - ئاگاداربە لە مارەکان! 🐍
5. بردنەوە - یەکەم یاریزان کە بگاتە خانەی 100 براوەیە 👑
